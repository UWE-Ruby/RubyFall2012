1. What is an object?
	everything you manipulate in Ruby is an object, starts out as an instance of a class, posesses its own unique set of instance varaiables, poseses a unique object identifier

2. What is a variable?
	a reference to an object, doesn't have to be declared, just pops into existence when named

3. What is the difference between an object and a class?
	a class is a combination of a state and methods that use that state, and object is an instance of that class

4. What is a String?
	sequences of characters, anything that's held within string delimiters: can be binary, letters, numbers, whatever
