1. What is a symbol?
An identifier corresponding to a string of characters.
2. What is the difference between a symbol and a string?
Symbols are references to a corresponding string of characters, while strings
are an object to class type String.  Strings can be altered while symbols can't be changed
3. What is a block and how do I call a block?
A block is a chunk of code between either braces or do end keywords that can be
invoke by associating it with a ruby iterator method.
4. How do I pass a block to a method? What is the method signature?
You can pass a block to a method by placing the code after the invocation of some method.
5. Where would you use regular expressions?
A regular expression is a pattern that can be used against a string.  It
can be used for to testing strings to match a pattern, extract and substitute
strings that matches a certain pattern.
