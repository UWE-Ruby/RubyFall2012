Chapter 10 Basic Input and Output
The Rake Gem: http://rake.rubyforge.org/

1. How does Ruby read files?
It uses the base class IO to handle input and output.  This base class is subclassed by other classes that provide more specific functionality.

2. How would you output "Hello World!" to a file called my_output.txt?
File.open("my_output.txt", w) do |f|
	f.puts "Hello world!"
end

3. What is the Directory class and what is it used for?
An I/O stream that deals with directories rather than individual files.  It helps you navigate directories and their contents.

4. What is an IO object?
An input/output channel between your ruby code and another file.

5. What is rake and what is it used for? What is a rake task?
It's a build tool written in ruby, used for automated building of projects and handling dependencies for you.  A rake task is a dependancy declaration.
