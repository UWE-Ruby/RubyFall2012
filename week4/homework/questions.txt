1. How does Ruby read files?
Ruby can read files using several methods including gets or IO.
2. How would you output "Hello World!" to a file called my_output.txt?
File.open("myoutput.txt", "w") { |file| file.puts "Hello World!"} end
3. What is the Directory class and what is it used for?
Objects of class Dir are directory streams representing directories in the underlying file systems. They provide a way to list directories and their contents.
4. What is an IO object?
Ruby defines a single base class, IO to handle input and output. 
5. What is rake and what is it used for? What is a rake task?
Rake is used for building code. Lets say you have a program that requires several other programs to be run first, you run your rack file with all the dependancies mapped, and your build is complete.
